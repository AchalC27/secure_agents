## Overview

The Identity layer provides decentralized identity management for AI agents.

```python
from arbiter import Identity
```

---

## High-Level API

### Identity Class

Convenience class for accessing identity components.

<ResponseField name="create_key_manager" type="method">
  Create a new key manager for generating and storing keys.
  
  ```python
  key_manager = Identity.create_key_manager()
  ```
</ResponseField>

<ResponseField name="create_issuer" type="method">
  Create a credential issuer.
  
  **Parameters:**
  - `issuer_did` (str): DID of the issuer
  
  ```python
  issuer = Identity.create_issuer("did:arbiter:issuer")
  ```
</ResponseField>

<ResponseField name="create_verification_hub" type="method">
  Create a verification hub for validating presentations.
  
  ```python
  hub = Identity.create_verification_hub()
  ```
</ResponseField>

<ResponseField name="create_revocation_manager" type="method">
  Create a revocation manager.
  
  ```python
  revocation = Identity.create_revocation_manager()
  ```
</ResponseField>

## DID Module

### DID

```python
from arbiter import DID
```

<ResponseField name="from_public_key" type="classmethod">
  Create DID from public key bytes.
  
  **Parameters:**
  - `public_key_bytes` (bytes): Raw public key
  
  **Returns:** `DID`
  
  ```python
  did = DID.from_public_key(auth_key.public_key.public_key_bytes)
  print(did.did_string)  # did:arbiter:2N4Q...
  ```
</ResponseField>

<ResponseField name="did_string" type="property">
  The full DID string.
  
  **Returns:** `str`
</ResponseField>

<ResponseField name="method" type="property">
  The DID method (always "arbiter").
  
  **Returns:** `str`
</ResponseField>

## Credential Issuance

### VCIssuer

```python
from arbiter.identity import VCIssuer
```

<ResponseField name="create" type="classmethod">
  Create an issuer with the given DID.
  
  **Parameters:**
  - `issuer_did` (str): DID of the issuer
  
  **Returns:** `VCIssuer`
</ResponseField>

<ResponseField name="issue_agent_identity_credential" type="method">
  Issue an agent identity credential.
  
  **Parameters:**
  - `subject_did` (str): DID of the subject
  - `agent_name` (str): Name of the agent
  - `agent_type` (str): Type of agent
  - `capabilities` (list[str]): Agent capabilities
  - `expiration` (datetime, optional): Expiration time
  
  **Returns:** `IssuedCredentialBundle`
  
  ```python
  bundle = issuer.issue_agent_identity_credential(
      subject_did="did:arbiter:agent",
      agent_name="ResearchBot",
      agent_type="researcher",
      capabilities=["search", "analyze"],
  )
  ```
</ResponseField>

<ResponseField name="issue_credential" type="method">
  Issue a custom credential.
  
  **Parameters:**
  - `request` (CredentialRequest): The credential request
  
  **Returns:** `IssuedCredentialBundle`
</ResponseField>

### IssuedCredentialBundle

<ResponseField name="credential" type="property">

</ResponseField>

<ResponseField name="witness" type="property">

</ResponseField>

<ResponseField name="handler_element" type="property">

</ResponseField>

<ResponseField name="raw_signature" type="property">

## Proof Generation

### ProofGenerator

```python
from arbiter.identity import ProofGenerator
```

<ResponseField name="__init__" type="constructor">
  Create a proof generator.
  
  **Parameters:**
  - `credential` (VerifiableCredential): The credential
  - `bbs_signature` (BBSSignature): Raw BBS+ signature

## Verification

### VerificationHub

```python
from arbiter.identity import VerificationHub
```

<ResponseField name="verify_presentation" type="method">
  Verify a presentation.
  
  **Parameters:**
  - `presentation` (VerifiablePresentation): The presentation
  - `expected_challenge` (str): Expected challenge
  - `expected_domain` (str, optional): Expected domain
  - `issuer_public_key` (BBSPublicKey): Issuer's key
  - `accumulator_value` (int): Current accumulator
  
  **Returns:** `VerificationResult`
  
  ```python
  result = hub.verify_presentation(
      presentation=presentation,
      expected_challenge=challenge,
      expected_domain="service.example.com",
      issuer_public_key=issuer_pk,
      accumulator_value=acc_value,
  )
  
  if result.is_trusted:
      print("Trusted!")
  ```
</ResponseField>

### VerificationResult

<ResponseField name="is_trusted" type="property">
  Whether the presentation is trusted.
  
  **Returns:** `bool`
</ResponseField>

<ResponseField name="decision" type="property">
  The trust decision.
  
  **Returns:** `TrustDecision`
</ResponseField>

<ResponseField name="disclosed_attributes" type="property">
  Attributes that were disclosed.
  
  **Returns:** `dict`
</ResponseField>

<ResponseField name="failure_reason" type="property">
  Reason for failure (if any).
  
  **Returns:** `str | None`
</ResponseField>
