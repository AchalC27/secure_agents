Learn how to issue, manage, and revoke verifiable credentials for AI agents.

## Overview

Credentials in Arbiter:
- Are signed with BBS+ for selective disclosure
- Include revocation witnesses
- Can assert any claims about subjects

---

## Setting Up an Issuer

```python
from arbiter import Identity

# Create an issuer with unique DID
issuer = Identity.create_issuer("did:arbiter:my-issuer")

print(f"Issuer DID: {issuer.did}")
print(f"BBS+ public key ready")
print(f"Revocation manager initialized")
```

---

## Credential Types

### Agent Identity Credential

The primary credential type for agent identity:

```python
bundle = issuer.issue_agent_identity_credential(
    subject_did="did:arbiter:agent",
    agent_name="ResearchBot",
    agent_type="researcher",
    capabilities=["search", "analyze"],
)
```

### Capability Credential

For granting specific capabilities:

```python
bundle = issuer.issue_capability_credential(
    subject_did="did:arbiter:agent",
    capabilities=["admin", "deploy", "configure"],
    scope="production-cluster",
    expiration=datetime.now(UTC) + timedelta(days=30),
)
```

### Custom Credentials

Issue credentials with any claims:

```python
from arbiter.identity import CredentialRequest

request = CredentialRequest(
    subject_did="did:arbiter:agent",
    credential_type="CustomCredential",
    claims={
        "clearanceLevel": 3,
        "department": "research",
        "projectIds": ["proj-123", "proj-456"],
    },
)

bundle = issuer.issue_credential(request)
```

## Credential Bundle

Every issued credential comes with a bundle:

```python
bundle = issuer.issue_agent_identity_credential(...)

# The credential itself
credential = bundle.credential
print(f"ID: {credential.id}")
print(f"Type: {credential.type}")
print(f"Subject: {credential.credential_subject.id}")
print(f"Claims: {credential.credential_subject.claims}")

# For revocation proofs
witness = bundle.witness
print(f"Witness epoch: {witness.epoch}")

# For accumulator
handler_element = bundle.handler_element

# For selective disclosure
raw_signature = bundle.raw_signature
```

## Credential Structure

```python
VerifiableCredential(
    context=["https://www.w3.org/2018/credentials/v1"],
    id="urn:aegis:cred:abc123",
    type=["VerifiableCredential", "AgentIdentityCredential"],
    issuer="did:arbiter:my-issuer",
    issuance_date=datetime(2024, 1, 15, 10, 30, 0, tzinfo=UTC),
    expiration_date=datetime(2025, 1, 15, 10, 30, 0, tzinfo=UTC),  # Optional
    credential_subject=CredentialSubject(

        claims={
            "agentName": "ResearchBot",
            "agentType": "researcher",
            "capabilities": ["search", "analyze"],
        }
    ),
    proof=CredentialProof(
        type="BbsBlsSignature2020",
        created=datetime(2024, 1, 15, 10, 30, 0, tzinfo=UTC),
        verification_method="did:arbiter:my-issuer#bbs-key",
        proof_purpose="assertionMethod",
    ),
)
```

## Revoking Credentials

When a credential should no longer be valid:

```python
# Revoke using the handler ID from the bundle
handler_id = bundle.handler_id

state = issuer.revocation_manager.revoke_credential(handler_id)

print(f"Credential revoked at epoch: {state.epoch}")
```

### Bulk Revocation

```python
# Revoke multiple credentials
handler_ids = ["handler:abc", "handler:def", "handler:ghi"]

for handler_id in handler_ids:
    issuer.revocation_manager.revoke_credential(handler_id)

print(f"Revoked {len(handler_ids)} credentials")
```

## Verification Integration

Issued credentials integrate with verification:

```python
from arbiter.identity import ProofGenerator, create_proof_request

# Holder creates presentation
generator = ProofGenerator(
    credential=bundle.credential,
    bbs_signature=bundle.raw_signature,
    witness=bundle.witness,
)

presentation = generator.generate_presentation(
    request=create_proof_request(
        challenge="verifier-challenge",
        required_attributes=["agentType"],
    ),
    issuer_public_key=issuer.bbs_keypair.public_key,
    accumulator_value=issuer.revocation_manager.get_current_accumulator(),
)

# Verifier checks
hub = Identity.create_verification_hub()
result = hub.verify_presentation(
    presentation=presentation,
    expected_challenge="verifier-challenge",
    issuer_public_key=issuer.bbs_keypair.public_key,
    accumulator_value=issuer.revocation_manager.get_current_accumulator(),
)
```

## Best Practices

<AccordionGroup>
  <Accordion title="Credential Expiration">
    Set appropriate expiration times:
    
    ```python
    bundle = issuer.issue_agent_identity_credential(
        subject_did=did,
        expiration=datetime.now(UTC) + timedelta(days=365),
        ...
    )
    ```
  </Accordion>
  
  <Accordion title="Minimal Claims">
    Only include necessary claims:
    
    ```python
    # Good - minimal claims
    claims = {"role": "researcher"}
    
    # Avoid - too much information
    claims = {
        "role": "researcher",
        "ssn": "xxx-xx-xxxx",  # Don't include!
        "full_history": [...]   # Don't include!
    }
    ```
  </Accordion>
  
  <Accordion title="Track Issued Credentials">
    Maintain a registry of issued credentials:
    
    ```python
    issued_credentials = {}
    
    def issue_and_track(request):
        bundle = issuer.issue_credential(request)
        issued_credentials[bundle.credential.id] = {
            "handler_id": bundle.handler_id,
            "subject": request.subject_did,
            "issued_at": datetime.now(UTC),
        }
        return bundle
    ```
  </Accordion>
</AccordionGroup>
